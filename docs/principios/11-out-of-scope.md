# 11 — Out-of-scope é declarado, não improvisado

> Princípio bônus §8.11. Fonte: [[PlanoBackend]].

## Regra

Tier 3 (contencioso, holding, planejamento avançado) **nunca** entra no produto. O assistente IA detecta e **encaminha ao marketplace**, nunca tenta responder. O ToS explicita os limites. ADR-0013 documenta cada categoria out-of-scope com justificativa.

## Cobertura deliberada

- Contencioso fiscal: ~15% → marketplace
- Eventos societários: 0% → marketplace
